# Personal Dashboard

这是一个仿照 Bento 风格设计的个人主页，包含深色/浅色模式切换、毛玻璃效果和动态交互组件。

## 功能特性

- **响应式布局**：基于 CSS Grid，适配不同屏幕尺寸。
- **深色/浅色模式**：
  - 支持自动检测系统主题。
  - 右上角按钮手动切换。
  - 带有平滑的过渡动画。
- **毛玻璃效果 (Glassmorphism)**：使用 `backdrop-filter` 实现现代半透明视觉效果。
- **动态组件**：
  - **时钟**：实时显示当前时间。
  - **日历**：自动生成当月日历并高亮当天。
  - **问候语**：根据时间段（早/中/晚）自动变化。
  - **音乐播放器**：点击播放按钮有简单的旋转动画交互（模拟）。
- **动态背景**：渐变背景自动流转动画。

## 如何使用

1. 直接双击打开 `index.html` 文件即可在浏览器中预览。
2. 确保电脑连接网络，因为项目使用了 CDN 加载图标库 (FontAwesome) 和随机头像 (DiceBear)。

## 文件结构

- `index.html`: 页面结构
- `style.css`: 样式定义（包含 CSS 变量主题配置）
- `script.js`: 交互逻辑（时钟、日历、主题切换）

## 自定义

可以在 `style.css` 的 `:root` 变量中修改颜色主题。
可以在 `index.html` 中修改链接和文本内容。
