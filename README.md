# Cyberpunk-Themed Interactive Personal Homepage

这是一个具有未来感和赛博朋克风格的交互式个人主页。项目在仿 Bento/iOS 质感设计的基础上，融入了大量动态交互效果，旨在提供独特而精致的视觉体验。

## ✨ 核心交互特性

- **磁力吸附交互 (Magnetic Interaction)**: 页面中的核心元素（如链接卡片）会像磁铁一样被光标吸引，当鼠标靠近时，元素会平滑地“吸”向光标，创造出一种有趣的物理吸引力。

- **文字“解码”动效 (Text Scramble Effect)**: 当鼠标悬停在链接卡片上时，卡片标题会经历一个快速的字符乱码、重组、最终“解码”成原文的动画，增添了浓厚的赛博朋克风味。

- **辉光边框 (Glowing Border)**: 鼠标悬停在卡片上时，会出现一个与其主题色一致的辉光边框，增强了视觉反馈。

- **故障扫描入场动画 (Glitch Scan-In Animation)**: 页面加载时，所有卡片会以一种“故障扫描”的效果逐个入场，仿佛信息正在被加载和渲染。

- **交互式背景粒子 (Interactive Particles)**: 背景中漂浮着动态粒子。当用户在页面上点击时，会从点击处产生一次粒子“爆炸”效果，增加了页面的互动性和趣味性。

- **高性能动画循环**: 所有动态效果（磁力、粒子、视差等）都被整合进一个统一的 `requestAnimationFrame` 循环中，确保在实现炫酷视觉的同时，页面依然保持绝对流畅。

## 🚀 基础功能

- **响应式布局**: 基于 CSS Grid，完美适配桌面和移动设备。
- **深色/浅色模式**: 支持自动检测系统主题，并可通过右上角按钮手动切换，过渡平滑。
- **毛玻璃效果 (Glassmorphism)**: 使用 `backdrop-filter` 实现现代半透明视觉效果。
- **动态组件**:
  - **翻页时钟**: 实时显示当前时间，字体为 `DS-Digital`。
  - **日历**: 自动生成当月日历并高亮当天。
  - **问候语**: 根据时间段（早/中/晚）自动变化。

## 🛠️ 技术栈

- **HTML5 / CSS3**: 页面结构与样式。
- **JavaScript (ES6+)**: 负责所有交互逻辑和动态效果。
- **Canvas 2D API**: 用于实现高性能的背景粒子效果。
- **CSS 自定义属性 (Variables)**: 用于主题化、动画参数控制和样式管理。
- **`requestAnimationFrame`**: 驱动所有平滑的动画效果。

## 🔧 自定义

- **内容修改**: 直接在 `index.html` 中修改链接、文本和个人信息。
- **样式与主题**: 在 `css/style.css` 的 `:root` 和 `.dark-theme` 规则中修改颜色、字体等基础样式。
- **交互参数**: 在 `js/script.js` 文件顶部的 `CONFIG` 对象中，可以方便地调整如磁力强度、粒子效果、动画速度等高级参数。

## 💡 未来美化构思

在当前基础上，还可以从以下几个方面进一步提升页面的视觉和交互体验：

1.  **音频反馈 (Audio Feedback)**:
    *   为关键交互（如点击粒子爆炸、文字解码完成、卡片入场）添加细微、科幻风格的音效，可以极大地增强沉浸感。

2.  **全局 3D 视差 (Global 3D Parallax)**:
    *   增强现有的视差效果，当鼠标在页面上大范围移动时，让所有卡片作为一个整体，在 Z 轴上产生更强烈的 3D 倾斜和位移，模拟一种悬浮在太空中的感觉。

3.  **多层星空背景 (Layered Starfield Background)**:
    *   将当前的动态背景升级为一个更深邃、带有视差效果的多层星空背景。可以加入随机划过的流星或星云动画，增加背景的深度和动态感。

4.  **创意主题切换 (Creative Theme Transition)**:
    *   在切换深色/浅色模式时，设计一个更具创意的过渡动画，例如“屏幕擦除”、“像素化溶解”或“圆形扩展”的效果，替代简单的颜色渐变。
